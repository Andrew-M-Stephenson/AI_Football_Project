## Activate .venv ##

.venv\Scripts\Activate.ps1

----------------------------------------

Controls:

Move between players in that frame:

D or → : next player

A or ← : previous player

Assign roles:

1 = QB

2 = RB

3 = WR

4 = TE

5 = OL

6 = DL

7 = LB

8 = CB

9 = S

Save and go to next image: S

Go back to previous image: B

Quit early: Q or Esc (it still saves everything you’ve labeled so far to data/meta/positions.json)

---------------------------------
